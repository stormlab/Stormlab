The MCImageManager PHP is a combination of PHP, HTML and Javascript, possible to integrate with the MCFileManager.
The purpose of the MCImageManager is to be able to preview images through thumbnails and conduct basic file operations
on them, such as upload and delete. The perfect tool to integrate with your CMS system, or TinyMCE.

For more information about MCImageManager, MCFileManager or TinMCE visit the TinyMCE website http://tinymce.moxiecode.com/.

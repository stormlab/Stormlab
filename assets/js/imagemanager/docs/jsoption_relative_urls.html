Option: relative_urls

If this option is set to true, all URLs returned from the MCImageManager will be relative from the specified document_base_url. This option is set to false by default.

Example of usage in init call on page level:

mcImageManager.init({
	relative_urls : true
});

Example of usage in mcImageManager.open call

<a href="javascript:mcImageManager.open('example1','url','','',{relative_urls : true});">[Browse]</a>

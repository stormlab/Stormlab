Option: dropdown.exclude_path_pattern

This is like an extra filer, you can filter the dropdown through this pattern to exclude folders from the dropdown list. Please be aware the you can get some rather strange behaviour regarding this option.

In short, this option is a regex match for the paths displayed in the dropdown selection.

Example of usage in the config.php file:

$mcImageManagerConfig['dropdown.exclude_path_pattern'] = "/admin/i";

Example of usage in a mc_access file:

dropdown.exclude_path_pattern=/admin/i

Option: thumbnail.gd.enabled

This is a boolean option to enable (true) or disable (false) to use the thumbnail functionality in MCImageManager, setting this to true, makes MCImageManager look for thumbnails, and depending on other settings, generate then on the fly, create them on upload, or just show them if they exist. Remember this option requires the GD module to be installed in PHP.

Example of usage in the config.php file:

$mcImageManagerConfig['thumbnail.gd.enabled'] = true;

Example of usage in a mc_access file:

thumbnail.gd.enabled=true

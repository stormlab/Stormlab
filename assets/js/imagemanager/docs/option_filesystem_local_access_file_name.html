Option: filesystem.local.access_file_name

This option enables you to specify the file name that is used to load directory specific ImageManager options.

Example of usage in the config.php file:

$mcImageManagerConfig['filesystem.local.access_file_name'] = "mc_access_file";

Example of usage in a mc_access file:

filesystem.local.access_file_name=mc_access_file

Option: general.toolbar

This option enables you to specify what tools are to be available in the toolbar for users. The option should contain a comma separated list of tools to be visible in toolbar. If you decide to use the filemanager as well, you need to configure filemanager.urlprefix option.

Available tools

  • refresh - Refresh view
  • upload - Upload file
  • createdir - Create directories
  • separator - Separator line
  • filemanager - Link to the filemanger (interoperability with the MCImageManager)

Example of usage in the config.php file:

$mcImageManagerConfig['general.toolbar'] = "refresh,upload,createdir";

Example of usage in a mc_access file:

general.toolbar=refresh,upload

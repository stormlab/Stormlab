Option: filemanager.urlprefix

Set the prefix path to the MCFileManager for automated integration. No trailing slash required. With this integration it is possible to "jump" between the MCImageManager and the MCFileManager.

Example of usage in the config.php file:

$mcImageManagerConfig['filemanager.urlprefix'] = "../filemanager";

Example of usage in a mc_access file:

filemanager.urlprefix=../filemanager

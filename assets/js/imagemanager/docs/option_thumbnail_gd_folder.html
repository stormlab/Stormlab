Option: thumbnail.gd.folder

This is a relative path to where the generated thumbnails should be stored and retrieved from, as default, it uses "mcith" folder. This folder will be created if it does not exist, the folder will be created as a subfolder to the folder where the image is.

If you do not wish to show this folder, you can filter it out using the filesystem.exclude_directory_pattern option.

Please note that this is a required configuration if you want to use the auto generate functionalty.

Example of usage in the config.php file:

$mcImageManagerConfig['thumbnail.gd.folder'] = "mcith";

Example of usage in a mc_access file:

thumbnail.gd.folder="mcith"

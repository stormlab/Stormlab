Option: dropdown.cache

This option enables you to cache the dropdown directory menu in the MCImageManager, this can increase performance on large sites, if you have a smaller website, you will not need to cache this.

Example of usage in the config.php file:

$mcImageManagerConfig['dropdown.cache'] = true;

Example of usage in a mc_access file:

dropdown.cache=true

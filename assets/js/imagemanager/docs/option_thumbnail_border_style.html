Option: thumbnail.border_style

The style on the border around the thumbnails. If there is more layout/design changes needed, the CSS and template file needs to be edited in the default theme.

Example of usage in the config.php file:

$mcImageManagerConfig['thumbnail.border_style'] = "1px solid #CCCCCC";

Example of usage in a mc_access file:

thumbnail.border_style=1px solid #333333

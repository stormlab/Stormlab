Option: thumbnail.gd.auto_generate

This is a boolean option to enable (true) or disable (false) to use the thumbnail autogenerate functionality in MCImageManager, setting this to true, makes MCImageManager genereate thumbnails if they do not exist already. Thumbnails are created in the folder as configured in thumbnail.gd.folder option.

Example of usage in the config.php file:

$mcImageManagerConfig['thumbnail.gd.auto_generate'] = true;

Example of usage in a mc_access file:

thumbnail.gd.auto_generate=true

Option: thumbnail.image_tools

Should contain a comma seperated list of tools to include on each image, such as preview, info and delete. The right corner icon is controled through another option called thumbnail.extension_image

Current options are:

  • preview
  • info
  • delete
  • edit

Example of usage in the config.php file:

$mcImageManagerConfig['thumbnail.image_tools'] = "preview,info,delete,edit";

Example of usage in a mc_access file:

thumbnail.image_tools=preview,info,edit

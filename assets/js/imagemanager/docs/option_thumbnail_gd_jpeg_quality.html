Option: thumbnail.gd.jpeg_quality

This is a 1-100 setting of the final quality for the jpeg thumbnails, no setting is avalible for thumbnails generated as png or gif. Setting this to 0 will use the default quality setting (75).

Recommended and default setting is 0 or 75.

Example of usage in the config.php file:

$mcImageManagerConfig['thumbnail.gd.jpeg_quality'] = 75;

Example of usage in a mc_access file:

thumbnail.gd.jpeg_quality=75

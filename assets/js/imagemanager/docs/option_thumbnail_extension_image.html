Option: thumbnail.extension_image

This is a boolean value that decides if the jpg,gif,png etc extension image should be shown in the upper right corner of each thumbnail image.

Example of usage in the config.php file:

$mcImageManagerConfig['thumbnail.extension_image'] = true;

Example of usage in a mc_access file:

thumbnail.extension_image=true

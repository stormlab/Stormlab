Option: thumbnail.information

This option should contain a comma seperated list of information to include in the image mouseover information window. It is not possible to change the order of their apperance, only if they should be included or not. Order in the comma seperated list does not matter.

These are the options avalible:

  • name
  • width
  • height
  • type
  • size
  • scale

Example of usage in the config.php file:

$mcImageManagerConfig['thumbnail.information'] = "name,width,height,type,size,scale";

Example of usage in a mc_access file:

thumbnail.information=width,height
